# 3. Lua Scripting Basics

Lua is the language Roblox uses. Here are the basics:

## Variables

```lua
local myName = "Alex"
```

## Printing

```lua
print("Hello, world!")
```

## If Statements

```lua
if myName == "Alex" then
    print("Hi Alex!")
else
    print("Who are you?")
end
```

## Loops

```lua
for i = 1, 5 do
    print(i)
end
```
